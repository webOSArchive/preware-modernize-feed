The proxy enables older apps (including the built-in browser) to use the modern encrypted Internet. After install, you need to launch them app, generate and install a cert, then turn the proxy on.

These MUST be installed with Preware or WOSQI in the order listed. Using the SDK command line will result in borked installs and much frustration.
